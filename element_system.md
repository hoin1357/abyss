# 속성 상성 시스템 설계 문서



## 기본 속성

### 1. 불(Fire)
- **특징**: 지속적인 화상 피해를 입힘
- **시각 효과**: 붉은색 불꽃 효과
- **상태이상**: 화상 - 2~3턴 동안 매 턴마다 추가 피해

### 2. 얼음(Ice)
- **특징**: 대상의 이동 및 공격 속도를 감소시킴
- **시각 효과**: 푸른색 얼음 결정 효과
- **상태이상**: 빙결 - 2~3턴 동안 이동 및 공격 속도 30% 감소

### 3. 풀(Nature)
- **특징**: 자가 회복과 적 약화 효과
- **시각 효과**: 연두색 생명력 효과
- **상태이상**: 약화 - 2~3턴 동안 공격력 25% 감소



## 속성 획득 및 적용

### 무기 속성

- 일부 인첸트 된 무기는 기본적으로 특정 속성을 가지고 있음
- 한 무기는 한 번에 하나의 속성만 가질 수 있음

### 스킬 속성

- 직업별 스킬은 고유한 속성을 가질 수 있음

## 속성 효과 시각화

- 상태이상 지속 시 해당 효과 아이콘 표시
