# 무기 시스템 및 자동 성장 설계 문서

## 무기 시스템 개요

'심연의 탐험가'의 무기 시스템은 다양한 무기 유형과 특성을 통해 전투의 다양성을 제공합니다. 각 무기는 공격속도, 데미지, 명중률의 차이를 가지며, 속성 부여를 통해 전략적 깊이를 더합니다.

## 무기 유형

### 1. 근접 무기
- **검(Sword)**: 균형 잡힌 성능 (중간 데미지, 중간 속도, 높은 명중률)
- **도끼(Axe)**: 높은 데미지, 느린 속도, 중간 명중률
- **둔기(Mace)**: 중간 데미지, 느린 속도, 방어력 관통 효과
- **단검(Dagger)**: 낮은 데미지, 매우 빠른 속도, 높은 치명타 확률

### 2. 원거리 무기
- **활(Bow)**: 중간 데미지, 중간 속도, 2-3칸 사거리
- **석궁(Crossbow)**: 높은 데미지, 느린 속도, 2-4칸 사거리
- **투척 무기(Throwing)**: 낮은 데미지, 빠른 속도, 1-2칸 사거리, 여러 개 소지 가능

### 3. 마법 무기
- **지팡이(Staff)**: 중간 마법 데미지, 중간 속도, 속성 피해 증가
- **완드(Wand)**: 낮은 마법 데미지, 빠른 속도, 마나 소모 감소
- **오브(Orb)**: 높은 마법 데미지, 느린 속도, 속성 효과 지속시간 증가

## 무기 특성

각 무기는 다음과 같은 기본 특성을 가집니다:

### 기본 특성
- **데미지(Damage)**: 기본 공격력
- **속도(Speed)**: 공격 속도 (턴당 공격 횟수에 영향)
- **명중률(Accuracy)**: 공격 성공 확률
- **사거리(Range)**: 공격 가능 거리 (근접: 1, 원거리: 2-4)
- **특수 효과(Special Effect)**: 일부 무기의 고유 효과

## 무기 등급

무기는 다음과 같은 등급으로 분류됩니다:

1. **일반(Common)**: 기본 성능, 흰색 텍스트
2. **고급(Uncommon)**: 기본보다 10-20% 향상된 성능, 초록색 텍스트
3. **희귀(Rare)**: 기본보다 30-50% 향상된 성능, 파란색 텍스트
4. **영웅(Epic)**: 기본보다 60-80% 향상된 성능, 보라색 텍스트
5. **전설(Legendary)**: 기본보다 100% 이상 향상된 성능, 주황색 텍스트

## 무기 강화 시스템

플레이어는 다음과 같은 방법으로 무기를 강화할 수 있습니다:

### 1. 강화석(Enhancement Stone)
- 무기의 기본 데미지, 속도, 명중률 증가
- 최대 +10까지 강화 가능
- 강화 단계가 높아질수록 성공 확률 감소

### 2. 속성 부여(Elemental Infusion)
- 무기에 속성(불, 얼음, 번개, 독, 풀) 부여
- 속성 결정(Elemental Crystal)을 사용하여 부여
- 한 무기에는 한 번에 하나의 속성만 부여 가능

### 3. 룬 각인(Rune Engraving)
- 특수 효과를 가진 룬을 무기에 각인
- 무기당 최대 3개의 룬 슬롯
- 룬 효과 예시:
  - 생명력 흡수: 데미지의 5-10% 체력 회복
  - 관통: 대상 방어력 10-20% 무시
  - 신속: 공격 속도 10-15% 증가
  - 정밀: 명중률 10-20% 증가
  - 치명: 치명타 확률 10-15% 증가

## 자동 성장 시스템

플레이어는 레벨업 시 자동으로 스탯이 증가하며, 직업에 따라 증가량이 달라집니다.

### 기본 스탯

1. **체력(HP)**: 생존 능력
2. **마나(MP)**: 스킬 사용 자원
3. **공격력(ATK)**: 물리 공격 데미지
4. **방어력(DEF)**: 받는 물리 피해 감소
5. **마법력(MAG)**: 마법 공격 데미지
6. **마법 저항(RES)**: 받는 마법 피해 감소
7. **속도(SPD)**: 행동 순서 및 회피율
8. **행운(LUK)**: 치명타 및 아이템 발견 확률

### 직업별 스탯 성장

### 경험치 획득

- 적 처치 시 적의 레벨과 유형에 따라 경험치 획득
- 보스 처치 시 대량의 경험치 획득
- 퀘스트 완료 시 추가 경험치 획득

### 레벨업 요구 경험치

레벨업에 필요한 경험치는 다음 공식으로 계산됩니다:
```
필요 경험치 = 기본값(100) × (현재 레벨 × 1.5)
```

예시:
- 레벨 1 → 2: 100 경험치
- 레벨 2 → 3: 300 경험치
- 레벨 3 → 4: 450 경험치
- 레벨 4 → 5: 600 경험치

### 최대 레벨

- 기본 최대 레벨: 20
- 특수 아이템을 통해 최대 레벨 30까지 확장 가능

## 스킬 포인트 시스템

레벨업 시 스킬 포인트를 획득하여 직업 스킬 트리에 투자할 수 있습니다.

### 스킬 포인트 획득
- 레벨업 시 1 포인트 획득
- 특정 보스 처치 시 추가 포인트 획득 가능
- 희귀 아이템을 통해 추가 포인트 획득 가능

### 스킬 해금 조건
- 필요 스킬 포인트 소모
- 선행 스킬 습득 필요 (트리 구조)
- 일부 스킬은 특정 레벨 도달 필요

### 스킬 강화
- 같은 스킬에 여러 포인트 투자 가능
- 투자한 포인트에 따라 스킬 효과 강화
- 최대 5레벨까지 강화 가능
