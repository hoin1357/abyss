# UI 및 UX 설계 문서

## 개요

'심연의 탐험가'의 UI(User Interface) 및 UX(User Experience)는 게임의 접근성과 시각적 매력을 높이기 위해 직관적이고 픽셀 아트 스타일에 맞는 디자인으로 구현됩니다. 이 문서는 게임의 주요 UI 요소와 사용자 경험 설계에 대한 가이드라인을 제공합니다.

## UI 디자인 원칙

1. **픽셀 아트 일관성**: 모든 UI 요소는 게임의 픽셀 아트 스타일과 일치하도록 디자인
2. **직관적 사용성**: 복잡한 메뉴 구조를 피하고 직관적인 조작 방식 제공
3. **정보 명확성**: 필요한 정보를 명확하고 간결하게 표시
4. **시각적 피드백**: 모든 사용자 액션에 대한 적절한 시각적/청각적 피드백 제공
5. **접근성**: 다양한 플레이어가 쉽게 이해하고 사용할 수 있는 디자인

## 색상 팔레트

UI 요소는 게임의 전반적인 색상 팔레트를 따르되, 다음과 같은 기능별 색상 코드를 사용합니다:

- **기본 텍스트**: #FFFFFF (흰색)
- **강조 텍스트**: #FFCC00 (금색)
- **경고/위험**: #FF3300 (빨간색)
- **긍정/회복**: #33CC33 (녹색)
- **정보/마나**: #3399FF (파란색)
- **특수/희귀**: #CC66FF (보라색)
- **배경 패널**: #333333 (어두운 회색, 70% 투명도)
- **테두리**: #666666 (회색)

## 주요 UI 요소

### 1. 헤드업 디스플레이(HUD)

게임 플레이 중 항상 표시되는 기본 정보 패널입니다.

#### 구성 요소:
- **체력 바**: 플레이어의 현재/최대 체력 표시 (빨간색 게이지)
- **마나 바**: 플레이어의 현재/최대 마나 표시 (파란색 게이지)
- **경험치 바**: 현재 경험치와 다음 레벨까지의 진행도 표시 (금색 게이지)
- **미니맵**: 현재 층의 축소된 맵 표시 (우측 상단)
- **현재 층 표시**: 현재 위치한 던전 층 표시
- **상태 아이콘**: 버프/디버프 등 현재 상태 효과 아이콘 표시
- **현재 장착 무기**: 현재 장착 중인 무기 아이콘 및 내구도 표시
- **퀵슬롯**: 빠르게 사용 가능한 아이템 슬롯 (최대 5개)

#### 배치:
- **상단**: 현재 층 표시, 미니맵
- **하단**: 체력/마나/경험치 바, 상태 아이콘, 장착 무기, 퀵슬롯
- **좌측 상단**: 플레이어 레벨, 직업 아이콘

### 2. 인벤토리 시스템

플레이어가 소지한 아이템을 관리하는 화면입니다.

#### 구성 요소:
- **아이템 그리드**: 5x6 크기의 아이템 슬롯 그리드
- **장비 슬롯**: 현재 장착 중인 장비 표시 (무기, 방어구, 액세서리 등)
- **아이템 정보 패널**: 선택한 아이템의 상세 정보 표시
- **정렬/필터 버튼**: 아이템 정렬 및 필터링 옵션
- **골드 표시**: 현재 보유 중인 골드량 표시
- **무게/용량 표시**: 현재 소지품 무게와 최대 용량 표시

#### 기능:
- **드래그 앤 드롭**: 아이템 이동 및 장착
- **우클릭 메뉴**: 아이템 사용, 장착, 버리기 등의 옵션 제공
- **아이템 스택**: 동일 아이템 자동 중첩
- **아이템 비교**: 장비 아이템 선택 시 현재 장착 장비와 비교 정보 표시

### 3. 캐릭터 정보 화면

플레이어 캐릭터의 상세 정보와 스탯을 표시하는 화면입니다.

#### 구성 요소:
- **기본 정보**: 이름, 레벨, 직업, 경험치
- **스탯 정보**: 체력, 마나, 공격력, 방어력, 마법력, 마법 저항, 속도, 행운 등
- **속성 저항**: 각 속성(불, 얼음, 번개, 독, 풀)에 대한 저항력 표시
- **특수 능력**: 치명타 확률, 회피율 등 특수 능력치 표시
- **상태 효과**: 현재 적용 중인 버프/디버프 목록
- **업적/기록**: 처치한 적 수, 발견한 아이템 수 등 기록

#### 기능:
- **스탯 상세 설명**: 각 스탯에 마우스 오버 시 상세 설명 툴팁 표시
- **성장 그래프**: 레벨업에 따른 주요 스탯 성장 그래프 표시

### 4. 스킬 트리 화면

직업별 스킬 트리를 표시하고 스킬 포인트를 투자할 수 있는 화면입니다.

#### 구성 요소:
- **스킬 트리 그래프**: 시각적으로 표현된 스킬 연결 구조
- **스킬 아이콘**: 각 스킬을 대표하는 픽셀 아트 아이콘
- **스킬 레벨**: 현재 습득한 스킬의 레벨 표시
- **스킬 포인트**: 사용 가능한 스킬 포인트 표시
- **스킬 정보 패널**: 선택한 스킬의 상세 정보 및 효과 설명

#### 기능:
- **스킬 습득**: 클릭으로 스킬 포인트 투자
- **스킬 미리보기**: 습득하지 않은 스킬 효과 미리보기
- **요구사항 표시**: 스킬 습득에 필요한 선행 스킬 및 레벨 표시
- **스킬 초기화**: 투자한 스킬 포인트 초기화 (특수 아이템 필요)

### 5. 로그 시스템

게임 내 이벤트와 전투 정보를 표시하는 로그 창입니다.

#### 구성 요소:
- **로그 창**: 스크롤 가능한 텍스트 영역
- **로그 필터**: 전투/아이템/퀘스트 등 카테고리별 필터링 옵션
- **중요도 표시**: 중요 메시지 강조 표시

#### 기능:
- **자동 스크롤**: 새 메시지 추가 시 자동 스크롤
- **색상 코드**: 메시지 유형별 색상 구분
- **로그 저장**: 최근 100개 메시지 저장 및 조회 가능

### 6. 툴팁 시스템

게임 내 요소에 대한 추가 정보를 제공하는 팝업 창입니다.

#### 구성 요소:
- **제목**: 아이템/스킬/상태 이름
- **설명**: 상세 설명 텍스트
- **스탯/효과**: 수치화된 효과 정보
- **비교 정보**: 장비 아이템의 경우 현재 장착 아이템과 비교 정보
- **추가 정보**: 희귀도, 요구 조건 등 추가 정보

#### 기능:
- **자동 위치 조정**: 화면 경계에 따라 툴팁 위치 자동 조정
- **상황별 정보**: 컨텍스트에 따라 다른 정보 표시
- **확장 정보**: Shift 키 누르고 있을 때 추가 상세 정보 표시

### 7. 대화 시스템

NPC와의 대화 및 퀘스트 정보를 표시하는 인터페이스입니다.

#### 구성 요소:
- **대화창**: NPC 대사 표시 영역
- **NPC 초상화**: 대화 중인 NPC의 픽셀 아트 초상화
- **선택지**: 플레이어 응답 선택지 목록
- **퀘스트 정보**: 관련 퀘스트 정보 및 보상 표시

#### 기능:
- **자동 진행**: 클릭 또는 스페이스바로 대화 진행
- **선택지 효과**: 선택에 따른 대화 분기 및 결과 변화
- **퀘스트 수락/완료**: 대화 중 퀘스트 관련 액션 수행

### 8. 맵 및 미니맵

던전 탐험을 위한 맵 시스템입니다.

#### 구성 요소:
- **미니맵**: 현재 층의 축소된 맵 (우측 상단에 항상 표시)
- **전체 맵**: 'M' 키를 눌러 확인할 수 있는 전체 맵 화면
- **방 구분**: 방 유형별 색상 구분 (일반, 전투, 보물, 보스 등)
- **아이콘**: 특수 지점 표시 아이콘 (상점, NPC, 보스, 출구 등)
- **시야**: 플레이어가 탐험한 영역과 미탐험 영역 구분

#### 기능:
- **자동 탐색**: 미니맵 클릭으로 해당 위치 자동 탐색 경로 표시
- **메모**: 특정 위치에 메모 추가 가능
- **확대/축소**: 전체 맵에서 확대/축소 기능
- **층간 이동**: 전체 맵에서 다른 층 맵 조회 가능

### 9. 메인 메뉴 및 일시 정지 화면

게임 시작 및 일시 정지 시 표시되는 메뉴입니다.

#### 구성 요소:
- **타이틀 로고**: 게임 로고 및 타이틀 아트
- **메뉴 버튼**: 게임 시작, 이어하기, 설정, 종료 등 버튼
- **설정 패널**: 소리, 그래픽, 조작 설정 등 옵션
- **저장/불러오기**: 게임 저장 및 불러오기 인터페이스

#### 기능:
- **애니메이션**: 메뉴 전환 시 부드러운 애니메이션 효과
- **배경 효과**: 움직이는 배경 및 파티클 효과
- **키 안내**: 단축키 및 조작법 안내

## 화면 전환 및 상호작용

### 화면 전환
- **페이드 인/아웃**: 주요 화면 전환 시 부드러운 페이드 효과
- **슬라이드**: 메뉴 열기/닫기 시 슬라이드 효과
- **팝업**: 알림 및 중요 정보는 팝업으로 표시

### 키보드 및 마우스 상호작용
- **WASD/화살표 키**: 캐릭터 이동
- **마우스 클릭**: 아이템 선택, 메뉴 조작, 공격 대상 지정
- **숫자 키 1-5**: 퀵슬롯 아이템 사용
- **E**: 상호작용 (아이템 줍기, NPC 대화 등)
- **I**: 인벤토리 열기
- **C**: 캐릭터 정보 화면
- **K**: 스킬 트리 화면
- **M**: 전체 맵 보기
- **ESC**: 일시 정지 메뉴

## 접근성 기능

- **텍스트 크기 조절**: UI 텍스트 크기 조절 옵션
- **색맹 모드**: 색맹 사용자를 위한 대체 색상 팔레트
- **키 재매핑**: 사용자 지정 키 설정 가능
- **자동 전투 옵션**: 반복적인 전투 자동화 옵션
- **튜토리얼 팁**: 초보자를 위한 게임 요소 설명 팁

## 시각적 피드백

- **피해량 표시**: 전투 시 피해량 숫자 팝업 표시
- **아이템 획득 알림**: 중요 아이템 획득 시 화면 상단에 알림 표시
- **레벨업 효과**: 레벨업 시 화려한 파티클 효과 및 사운드
- **위험 경고**: 체력 낮을 때 화면 가장자리 붉은 효과
- **상태이상 효과**: 캐릭터 주변에 상태이상 시각 효과 (불꽃, 얼음 등)

## 모바일 대응 (향후 확장)

- **터치 컨트롤**: 가상 조이스틱 및 액션 버튼
- **제스처**: 스와이프로 메뉴 전환, 핀치로 확대/축소
- **자동 크기 조절**: 다양한 화면 크기에 맞춰 UI 자동 조정
- **세로 모드 지원**: 세로 화면 레이아웃 최적화

## UI 애셋 목록

1. **아이콘 세트**
   - 아이템 카테고리 아이콘 (무기, 방어구, 소비품 등)
   - 스탯 아이콘 (체력, 마나, 공격력 등)
   - 상태이상 아이콘 (화상, 빙결, 중독 등)
   - 스킬 아이콘 (직업별 스킬 표현)

2. **프레임 및 패널**
   - 기본 UI 프레임 (다양한 크기)
   - 팝업 창 배경
   - 버튼 (일반, 호버, 클릭, 비활성 상태)
   - 스크롤바 및 슬라이더

3. **게이지 및 바**
   - 체력/마나 바 (다양한 길이)
   - 경험치 바
   - 진행 바
   - 쿨다운 인디케이터

4. **폰트**
   - 픽셀 아트 스타일 기본 폰트
   - 강조 텍스트용 대형 폰트
   - 숫자 전용 폰트 (데미지 표시용)

## 구현 우선순위

1. **핵심 HUD**: 체력/마나/경험치 바, 현재 층 표시
2. **로그 시스템**: 게임 이벤트 및 전투 정보 표시
3. **인벤토리 시스템**: 아이템 관리 및 장비 장착
4. **캐릭터 정보 화면**: 스탯 및 상세 정보 표시
5. **툴팁 시스템**: 아이템 및 스킬 정보 표시
6. **스킬 트리 화면**: 스킬 습득 및 관리
7. **미니맵**: 던전 탐험 보조
8. **대화 시스템**: NPC 상호작용
9. **메인 메뉴 및 설정**: 게임 시작 및 환경 설정
