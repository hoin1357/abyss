# abyss

모바일 웹에서 실행되는 턴제 로그라이크 프로토타입입니다.

## 실행

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173` 접속.

## 조작

- 맵 영역 탭/클릭: 해당 타일로 자동 이동
- 적 탭/클릭: 접근 후 공격
- 하단 통합 UI 버튼:
  - ⏱ 대기
  - P 회복 포션 사용
  - S 저장
  - L 불러오기

## 반영된 핵심 변경

- 상단 HUD + 게임 영역 + 하단 조작부를 하나의 캔버스 UI로 통합
- 어두운 석재/수로 기반 픽셀 톤으로 렌더링 스타일 개선
- 유닛 스프라이트를 타일의 약 83%(`40px / 48px`)로 확대
- 턴제 전투, 시야, 허기, 층 이동, 세이브/로드 유지

- 픽셀 수를 높인 고해상도 타일/스프라이트 렌더링
- 첨부 레퍼런스에 맞춘 벽돌 벽/석재 바닥/방 수로 그래픽 반영
